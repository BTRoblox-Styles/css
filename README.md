# BTRoblox CSS

This repository contains the primary and secondary CSS files used by the BTRoblox extension. These styles power the UI tweaks, layout fixes, and design improvements BTRoblox brings to the Roblox website.

## Files

- `primary.css` – Core styles used by BTRoblox
- `secondary.css` – Additional or optional styling enhancements

## Usage

These CSS files are part of the BTRoblox extension's customization layer. You can use them as-is or modify them for your own fork or version of the extension.
